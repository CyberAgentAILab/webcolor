## Download dataset

```bash
./download.sh {main,cache,text,image}
```

    -   Main dataset
    -   Split file
    -   Note: Processing to DGL format takes about 50 minutes.

-   "cache" (607 MB, recommended)

    -   Processed DGL cache

-   "text" (356 MB, optional)

    -   Text data for reference

-   "image" (6.3 GB, optional)

    -   Image data for reference
